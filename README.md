Fake your Roblox profile locally. Press + to open menu.

Usage Tested on Chrome, Firefox, LibreWolf, Edge. Recommended Firefox + Violentmonkey.

Install Violentmonkey https://violentmonkey.github.io/get-it/ or Tampermonkey (for chrome and edge) https://www.tampermonkey.net/index.php Download usescript and click create new script in violent or tampermonkey then paste in and save as. On roblox, press + to display the menu.
